# visor-camarasIP-nvr
Este proyecto es un visor de cámaras IP que se conecta a un NVR, mediante el protocolo RTSP, permitiendo la visualización en tiempo real de múltiples cámaras desde una PC en la misma red local.  El objetivo es ofrecer una alternativa ligera y libre de licencias a softwares propietarios como iVMS-4200, evitando limitaciones.
